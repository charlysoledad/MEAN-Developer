## What is Node.js
- Runtime environment for server-side and networking applications
- Open Source
- Cross platform
- Built on top Chrome's JavaScript runtime, the V8 engine
- Node applications are written in JavaScript
- Lighweight
+ #### Used by:
  + Netflix
  + Paypal
  + Google
  + Github
  + LinkedIn, etc
